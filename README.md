# parrot_pirot
飛行機に搭乗するまでの旅行気分を味わいたいと思って作りました

![travel](https://user-images.githubusercontent.com/52645663/104793102-5da22700-57e4-11eb-982c-6b07b6b6a90a.gif)

## セットアップ
### リポジトリクローン
```
git clone https://github.com/ayaka-ramens/parrot_pirot.git
```

### ライブラリをインストール
```
npm install
```

### 開発環境での実行
```
npm run serve
```
